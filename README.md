# 基于Python的Django+Vue学生成绩管理系统 python749

## 项目概述

本项目是基于Python语言，采用Django框架与Vue.js构建的学生成绩管理系统。系统旨在提供一种高效、便捷的成绩管理解决方案，满足学校、教师及学生对成绩管理的需求。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统的主要功能模块包括：

- **首页**：展示系统概览信息。
- **个人中心**：用户可管理个人信息。
- **教师管理**：对教师信息进行管理。
- **学生管理**：对学生信息进行管理。
- **公告信息管理**：发布和管理公告信息。
- **课程类型管理**：对课程类型进行管理。
- **课程信息管理**：对课程信息进行管理。
- **选课信息管理**：管理学生选课信息。
- **成绩信息管理**：录入、查询和管理学生成绩。

## 系统角色

系统主要角色包括：

- 管理员：负责系统的维护和用户管理。
- 教师：负责成绩录入和查询。
- 学生：查询个人成绩和选课信息。

## 运行环境

- Python环境：推荐使用Python 3.6及以上版本。
- 数据库环境：MySQL 5.7及以上版本。
- 前端环境：现代浏览器。

## 部署步骤

1. 环境准备：安装Python、MySQL、Node.js等依赖环境。
2. 数据库搭建：根据项目提供的SQL脚本初始化数据库。
3. 后端部署：运行Django项目。
4. 前端部署：构建Vue项目并运行。

## 目录结构

```
project/
├── backend/          # 后端代码目录
│   ├── apps/         # Django应用目录
│   └── manage.py     # 管理脚本
└── frontend/         # 前端代码目录
    ├── src/          # Vue源代码
    ├── public/       # 公共资源
    └── package.json  # Node.js包管理文件
```

- **高效性**：采用Python + Django技术栈，确保系统的高效运行。
- **易用性**：基于Vue.js构建的友好交互界面，易于操作和使用。
